The hdf5 Module¶

Utility functions for HDF5 files.

This module contains utility functions that manipulate HDF5 files (load whole or partial data blocks, write to normal or VDS files, etc.).

load_hdf5_data(hdf5_filename, hdf5_path)[source]¶

Loads a data block from an HDF5 file.

This function loads into memory the whole content of a data block located in an HDF5 file.

Parameters:
  • hdf_filename (str) – the relative or absolute path to an HDF5 file containing the data to load.
  • hdf5_path (str) – the internal HDF5 path to the data block to load.
Returns:

the data loaded from the HDF5 file.
Return type:

Any
Raises:

OndaHdf5FileReadingError – if any error occurs while reading the data from the file.
