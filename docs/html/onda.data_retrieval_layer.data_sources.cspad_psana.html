The cspad_psana Module¶

Retrieval of CSPAD detector data from psana.

This module contains functions that retrieve data from a CSPAD x-ray detector using the psana framework.

get_peakfinder8_info()[source]¶

Retrieves the peakfinder8 information for the CSPAD detector.

Returns:a named tuple storing the peakfinder8 information.
Return type:Peakfinder8Info
detector_data(event)[source]¶

Retrieves one frame of CSPAD detector data from psana.

Parameters:event (DataEvent) – an object storing the event data.
Returns:one frame of detector data.
Return type:numpy.ndarray
