class CrystallographyFrameViewer(geometry, hostname, port)[source]¶

Bases: onda.utils.gui.OndaGui

OnDA frame viewer for crystallography.

This viewer receives detector frame data from an OnDA crystallography monitor, when it is tagged with the ‘ondadetectordata’ label. It displays the received detector frames, together with any detected Bragg peak (if present). A data buffer allows the viewer to stop receiving data from the monitor but still keep in memory the last 10 displayed frames for inspection.

Parameters:
  • geometry (Dict[str, Any]) – a dictionary containing CrystFEL detector geometry information (as returned by the ‘load_crystfel_geometry` function in the ‘cfelpyutils’ module).
  • hostname (str) – the hostname or IP address where the viewer will listen for data.
  • port (int) – the port at which the viewer will listen for data.
